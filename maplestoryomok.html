Maplestory Omok
HTML CSS Materialize JavaScript Node.js Express Socket.IO

A recreation of the Omok minigame from Maplestory. Omok (오목 [五目]) is a 2-player traditional korean game similar to connect 4. Players take turns placing their respective pieces on a board and the first to connect 5 in a row horizontally, vertically, diagonally, or antidiagonally wins. The game is built with Node.js, Express, JavaScript, and designed with HTML, CSS, Materialize. Like in the actual minigame from Maplestory, there is a small chatbox for players to message each other.

Click to watch the demo video: https://www.youtube.com/watch?v=n4Epytr7pb0

Currently, the chatbox feature and 2-player game connection is being implemented with socket.IO.
